# Weather App

Name: Nompumelelo Tshabalala 
Student Number: ST10099914 

## Purpose
This app allows users to capture weekly weather data, calculate
average temperature, and view detailed results.

## Features
- Splash screen
- Main screen with input and average calculation
- Detail screen showing all days
- Uses arrays and loops
- Navigation between screens

## GitHub Link
https://github.com/mpumeleloMshengu/weather-app-imad

## Pseudocode

START

Create arrays for days, min, max, weather

User inputs data → store in arrays

Loop through arrays → calculate average

Display average

When user clicks View:
   send data to Detail screen

Detail screen:
   loop through arrays
   display all data

END
